Installe le module pour gérer le stockage local asynchrone (sessions, préférences, etc.)
npm install @react-native-async-storage/async-storage

Installe le composant Checkbox compatible avec Expo/React Native
npm install expo-checkbox

 (Optionnel) Installe les types TypeScript pour expo-checkbox (souvent inutile car déjà inclus)
npm install --save-dev @types/expo-checkbox
